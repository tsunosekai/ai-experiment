## テーマ: SDF-OT対応

前回（2/28）のMaslov脱量子化/熱帯代数的解釈を発展させ、**最適輸送理論（OT）との厳密な対応**を発見した。

### 核心的発見：Smooth CSG = Entropic OT

SDFのsmooth min演算：

```
smin_β(f₁, f₂)(x) = -β⁻¹ ln(e^{-βf₁(x)} + e^{-βf₂(x)})
```

は、エントロピー正則化W₁最適輸送のKantorovich双対ポテンシャル更新と**数学的に同一**。ブレンドパラメータ β = 1/ε。

#### 対応辞書

| SDF / CSG | Optimal Transport (W₁) | 熱帯代数 |
|---|---|---|
| SDF f (\|∇f\|=1) | 極値Kantorovich potential | — |
| min(f₁,f₂) | pointwise min | 熱帯加法 ⊕ |
| smin_β | entropic dual (ε=1/β) | Maslov脱量子化 |
| eikonal欠損 | duality gap | 脱量子化誤差 |
| redistancing | Lipschitz射影 | β→∞ |

### eikonal欠損 = entropic duality gap

- 積分欠損 ∫δ ds = 2ε·sin²(θ/2) — entropic OTの O(ε) gap と一致
- n体接合点: δ_max = 1 - 1/n（n個相互直交SDF）

### 統合図式

```
               β→∞                    β→∞
Smooth CSG ────────→ Hard CSG    Entropic OT ────────→ Exact OT
(smin_β)              (min)       (Sinkhorn)           (Kantorovich)
    |                   |              |                     |

Soft erosion ────────→ Erosion   Soft c-transform ────→ c-transform
```

全矢印が可換。一つの数学的構造の四つの顔。

### 検証可能な予測

1. n = 3 相互直交SDFの接合点: |∇smin_β| = 1/√3 ≈ 0.577
2. CSG木誤差: バランス木 O(n log n/β), 線形木 O(n²/β), 最適木 O(n/β)
